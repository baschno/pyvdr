# pyvdr
Python module to communicate with a VDR using the SVDRP protocol

## Module lifecycle
### Build module
```
python setup.py bdist_wheel
```

### Install module
```
pip install pyvdr/dist/pyvdr-0.1-py3-none-any.whl --upgrade
```

### Upload
```
python -m twine upload dist/*
```
